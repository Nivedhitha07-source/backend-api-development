# Backend API Development

## 📌 Overview

This project is a simple Backend API developed using Node.js and Express.js. It demonstrates how to create RESTful API endpoints, process client requests, validate user input, and return appropriate responses.

---

## 🚀 Features

- REST API Development
- GET & POST Endpoints
- Request and Response Handling
- Input Validation
- JSON Data Processing
- Error Handling
- Clean Code Structure

---

## 🛠 Technologies Used

- Node.js
- Express.js
- JavaScript
- REST API

## 💻 Installation

```bash
git clone https://github.com/your-username/backend-api-development.git
```

```bash
cd backend-api-development
```

```bash
npm install
```

```bash
npm start
```

Server runs on:

```
http://localhost:3000
```

---

## 📌 API Endpoints

### GET

```
GET /api/users
```

Returns all users.

### POST

```
POST /api/users
```

Adds a new user.
